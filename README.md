## Environment Prepare

Copy environment file

```bash
cp .env.local .env
```

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```

### Run story book

```bash
npm run storybook
```

### URLs

```bash
# View story book
http://localhost:6006

# visit site
http://localhost:3001
```
